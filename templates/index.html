License Plate Recognition API

Introduction

This RESTful API is developed using Flask. Flask is a Python
web framework, meaning it is a third-party Python library used for
developing web applications.

The API

The license plate recognition API provides a few endpoints that allows
the main application fetch records relating to any plate number in the
database. The API used a document format or NoSQL database to store data
in documents similar to JSON (JavaScript Object Notation) objects. This
choice of database is informed by the fact that NoSQL databases were
built with flexibility and scalability in mind.
