Etomi License Plate API documentation

Introduction

This RESTful API is developed using Flask. Flask is a Python web framework, meaning it is a third-party Python library used for developing web applications.

The API

The license plate recognition API provides a few endpoints that allow the main application fetch records relating to any plate number in the database. The API used a document format NoSQL database to store data in documents similar to JSON (JavaScript Object Notation) objects. This choice of database is informed by the fact that NoSQL databases were built with flexibility and scalability in mind when compared with relational databases.

Endpoints

Base URI: https://etomi-api.herokuapp.com

  1. ‘/’: returns the documentation page of the api (this page you are viewing now).
  2. /plates: return all Plate numbers in the database. HTTP request type is [GET]
  3. /plates/<plate>: returns all the data associated with a specific plate number. HTTP request type is [GET]
  4. /plates/add: adds a new plate number and its related data to the database. HTTP request type is [POST]
  5. /plates/<plate>/update: updates the records of the specified plate number. HTTP request type is [PUT]
  6. /plates/<plate>/delete: deletes the records of a specific plate number. HTTP request type is [DELETE]

Usage

Imports: import requests,json

  1. Ensure that requests is installed in your application otherwise, install it. Then import requests and json
  2. Make API call: response = requests.get(BaseURI + endpoint')

    For example, to request the information about a specific plate number: combine the base url https://etomi-api.herokuapp.com and the endpoint /plates/<plate>

    So your API call will look like this: response = requests.get('https://etomi-api.herokuapp.com/plates/ABC53GC')

    NOTE: the plate number should be passed as a variable not hardcoded like the example above. You can do this: 

            plate_number = 'ABC53GC'
        response = requests.get('https://etomi-api.herokuapp.com/plates/'+ plate_number)

  3. Convert response to JSON object: vehicle = response.json()

    The JSON object is a python dictionary and it looks like this:

            {
          car_type: "Toyota Camery",
          color: "Black",
          comment: "This plate has never been flagged",
          driver: "Onoriode Thompson",
          is_ban: 0,
          plate: "ABC53GC"
        }

    Use vehicle['car_type'] to access the the car type
